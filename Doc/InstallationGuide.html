Getting Started

  1. Install Nunit 2.6.3
  2. Copy the Concordion.Integration.NUnit.dll to C:\Program Files (x86)\NUnit 2.6.3\bin\addins
  3. Download the Concordion-kickstart Project
  4. Make sure that follwing dlls are linked (will be found in the Project under Libs):
    • concordion.dll
    • nunit.framework.dll
    • OGNL.dll
  5. You should find following lines in AssemblyInfo.cs of the project specs

  6. Check the properties for the HTML files

    7. Please follow our Installation Guide

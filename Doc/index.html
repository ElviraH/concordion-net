Concordion-NET

Concordion-NET is an open source tool for writing automated acceptance tests in C#. It supports Behavior Driven Development (BDD) similar to great tools such as Cucumber, SpecFlow, FitNesse, and NBehave. Based on the excellent classification of BDD tools, it focuses on business readable input.

Specification by Example

As Concordion specifications are written in simple HTML, it provides a good basis for a living documentation system. You can use a what-you-see-is-what-you-get (WYSIWYG) editor like BlueGriffon to describe your intended behavior of your new software.

Watch the video to see how to use Concordion-NET to write tests that are readable to business people and can help to achive the goal of purpose driven development. The concrete examples within the specifications are turned into executable specifications with a small set of commands (e.g. "set", "execute", "assertEquals"). These allow checking the examples against the real-life system.

Here are some additional information to start working with Concordion-NET:

  • On the Getting Started Page you'll find a kick-start solution for your C# projects in Visual Studio as well as a first example.
  • To setup your infrastructure, please download the latest version and follow the installation guide.
  • To run the test please visit running Concordion tests with Nunit

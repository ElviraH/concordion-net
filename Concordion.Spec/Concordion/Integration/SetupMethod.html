Using SetUp Attribute to Run Setup Methods befor Test Execution

NUnit provides a mechanism to call functions just before each test execution. This is accomplished with the help of the NUnit SetUpAttribute.

This mechanism can be used in Concordion.NET as well. All methods marked with the attribute "[SetUp]" are executed before the actual processing of the active specification.

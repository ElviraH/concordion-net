Read List

Example

Given the following HTML list:

  • item 1
  • item 2
    • sub item 1
    • sub item 2
  • item 3
    • sub item 3
      • sub sub item 1
    • sub item 4
  • item 4

When item 3 is printed in the list

All child items are included in the print as well: sub item 3, sub sub item 1, sub item 4
